Evanto Authors Portal to manage products and verify purchases

== Description ==
Manage your Evanto account in one place.
Create a token here https://build.envato.com/create-token/
Features:
Evanto Purchase Verification
My collections
My orders
Downloads

== Installation ==
- Upload to your WordPress site plugin and activate
- Go go https://build.envato.com/create-token/ and obtain a token
- Enter your token and your username
